About Me

Hello there! I'm Jeffrey Li. I am a third year computer science student at the University of Toronto, currently working towards an Honours Bachelors of Science in Software Engineering.

I have been passionate about computer science since middle school. I have participated in various coding challenges and hackathons, such as the ORBIS Challenge and RUHacks. Projects

TEQ-LIP Data Reporting Platform
  • Java
  • SQLite

A Java-based data reporting application designed for the Toronto East Quadrant Local Immigration Partnership (TEQ-LIP) to centralize data collected from local immigration agencies in the Scarborough area, and generate reports and summaries for the partnership.

Needham-Schroeder Protocol CLI
  • Python
  • Docker

A command-line interface for the Needham-Schroeder public-key protocol and its related MITM attack utilizing socket programming and RSA and AES from the Pycryptodome library. Docker was employed to simulate a network involving a basic file storage server, a malicious storage server, a naive client, and a public-key server.

Bundy Budgetting Application
  • Materialize
  • Firebase

A web service designed to help users in better managing their budget. The Materialize framework was used to develop the front end, while Firebase was used for back-end data storage and user authentication. (Built during THacks 2017)

SOLID Android Banking App
  • Java
  • SQLite
  • Android Studio

A software application that mimics the basic features of a bank. User are able to create their own Savings or Chequing accounts, make transactions, and send messages to each other. The app was developed for both desktop and mobile Android platforms.

Conway's Game of Life
  • Python

A Python applet that simulates Conway's Game of Life, implemented using the Pygame module. Users can run a simulation, modify the generation speed, and set custom rules for a cell's birth and survival.
